# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-04

### Added
- Initial release of Thready
- ThreadPool singleton for easy worker management
- WorkerPool class for direct pool management
- Support for transferable objects (zero-copy)
- Task queuing and load balancing
- Worker health monitoring and automatic recovery
- Pool statistics API
- Full TypeScript support with type definitions
- Examples for React, Vite, and vanilla JavaScript
- Comprehensive documentation

### Features
- Singleton pattern for global access
- Automatic worker recycling
- Smart queue management
- Error handling and recovery
- Support for multiple task types
- Configurable pool size
- Browser and bundler compatibility
